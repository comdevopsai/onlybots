# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this repository, please report it
**privately** rather than opening a public issue.

Send details to: **commondevops@proton.me**

Please include:
- A description of the vulnerability and its impact
- Steps to reproduce (if applicable)
- Any relevant logs, proofs of concept, or references

We will acknowledge receipt, work with you on a fix, and coordinate disclosure.
Do not disclose the issue publicly until a patch has been released.

Thank you for helping keep this project secure.
